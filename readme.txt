[center][size=12pt][u][b]Compatibility[/b][/u][/size][/center]

The "[i]Gender Image To Text[/i]" is compatible for only SMF [b]2.0.X[/b]. It will soon be updated for 1.1.X.

[center][size=12pt][u][b]Features[/b][/u][/size][/center]

The Gender Image To Text Modification changes all Gender Images on the website to text. This may be useful; for Mobile or WAP Forums, or mainly because most people can't tell the difference, or tell which ones the images are, as they are not used alot in today's society.

[center][size=12pt][u][b]Installation[/b][/u][/size][/center]

Install this modification through the package manager. It only edits one small part of a file, so nothing [i]should[/i] go wrong.
